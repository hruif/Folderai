What is it?

Everything before anything moves.

  • Local AI sorting Classifies loose files by type and subject, then suggests where they belong in your existing folders.
  • Content-aware Uses supported text, PDF, DOCX, and OCR content to understand more than just filenames.
  • Staged review Edit destinations, rename files, create folders, search, and drag items before executing.
  • Plain requests Tell it things like "put invoices in Finance" or "remove old installers" before cleanup.
  • Safe removal Duplicates, installers, incomplete downloads, and junk go to quarantine, not permanent deletion.
What's good about it?

Less anxiety around big cleanup jobs.

  • Private by default Local model work stays on your Mac. No account, analytics, or cloud file upload.
  • Built for corrections The plan is editable, and Folderai can learn from the destinations you choose.
  • Designed for real folders Works with your Downloads, Desktop, Documents, and added folders instead of forcing a fixed scheme.
